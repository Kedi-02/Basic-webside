Css layout

CSS er er for å få selve siden til å se så bra ut og oversiktelig som mulig. Mens HTML er selve informasjonen, så er CSS hvordan det skal se ut og hvor lett det er å lese informasjonen for nettlesere. Vi har for eksempel CSS flexbox og Grid som er de mest vanligste nå for tiden.

Flexbox gjør at man kan dele celler og gjøre de om til mindre/fler celler, altså i egne seksjoner hvor man kan ha forskjellig informasjon på. Flexbox, det er noe man driver med i mens man driver med Grid også.

Grid gjør det mye enklere for å struktrere en nettside. Grunnen til dette, er at man har egne celler hvor alt ligger på, samtidig har man også ruter, som mn i tilleg kan endre på. Dette er veldig produktivt å drive med, og sparer en god del tid og er en god måte å designe sin nettside.

Før i tiden, så drev man mye mer å justere ting på en manuell måte. Vi har for eksempel 'float' som ble brukt før Css Gird og Flexbox kom. På grunn av dette, så brukte man mye mer tid med selve kodingen før i tiden, enn i dag, fordi det var mer koding man måtte drive med, og man måtte bruke mer tid på å sørge for at innholdet og strukturen var i riktig plass.
